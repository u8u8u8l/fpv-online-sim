# fpv-online-sim
a webscraped version of [lightning8113](https://lightning8113.itch.io/)'s drone sim. all credit to him. this is not my project, this is simply the source

### INSTRUCTIONS FOR RUNNING
> [!NOTE]
> you must have python 3.x or higher installed to run this

```
git clone https://github.com/u8u8u8l/fpv-online-sim.git
cd fpv-online-sim
python3 -m http.server 8080
```

navigate to http://0.0.0.0:8080
